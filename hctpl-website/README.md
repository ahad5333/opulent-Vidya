# 🚀 HCTPL Website

A modern, animated, AI-focused company website built using **Next.js (App Router)**, **React**, **Tailwind CSS**, and **Framer Motion**.  
It includes smooth animations, interactive UI components, job listings, testimonials, and modular sections.

---

## ✨ Features

### 🎯 Core Sections
- Modern Landing Page
- Animated Navbar with active wave indicator
- Hero section with smooth transitions
- Services section
- Products section
- About section with image + 3D hover effects
- Logo marquee with spotlight hover animation
- Testimonials section (with video support)
- Careers page with job listings + modal popup
- Contact/Footer section

---

## 🎨 UI/UX Highlights
- Glassmorphism effects
- Gradient backgrounds
- Hover glow & spotlight effects
- Smooth Framer Motion animations
- Responsive design (mobile-first)
- Card tilt / hover interactions
- Modern SaaS-style UI

---

## 💼 Careers Module
- Job listing cards with:
  - Title
  - Location
  - Type (Full-time / Contract)
  - Job Description (JD)
  - Skills tags
- Click-to-open modal popup
- Apply button (UI ready for backend integration)
- Easy future admin integration for job posting system
